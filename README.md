## 🚀 Featured AI/ML Research & Projects

### 🧠 Advanced AI Research & Architectures

1. **[Hierarchical Metacognitive Code Generation (HMCG)](https://github.com/saddam1838/hmcg-multi-agent-codegen)**
   - **Description:** Multi-agent code generation pipeline using DeepSeek-V3.2 with PI/PE validation to catch structural symmetry bugs.
   - **Impact:** Achieved **92% execution success** (vs 58% baseline) and reduced debugging iterations by 56%.
   - **Tech:** Python, DeepSeek-V3.2, Hugging Face Transformers, Streamlit

2. **[Continuous-Time Convolutional Neural Operator (C-CNO)](https://github.com/saddam1838/c-cno-neural-operator)**
   - **Description:** Reproduced C-CNO architecture in PyTorch for time-dependent PDEs (Navier-Stokes) using ReNO-compliant spatial processing and FiLM conditioning.
   - **Impact:** Demonstrated stable inference and competitive relative L1 errors against 8 baselines.
   - **Tech:** Python, PyTorch, Neural Operators, FiLM

3. **[Neuron-Guided Cross-Lingual Alignment (NG-CLA)](https://github.com/saddam1838/ng-cla-llm-interpretability)**
   - **Description:** Investigated mechanistic interpretability in BLOOM-560M. Applied targeted LoRA fine-tuning to 1,133 shared semantic neurons.
   - **Impact:** Boosted Spanish Cross-Retrieval Ratio (XRR) by **3×** (to 0.200) on FLORES-101.
   - **Tech:** Python, PyTorch, Hugging Face Transformers, LoRA

4. **[Random Sparse Attention Transformer for Object Detection](https://github.com/saddam1838/sparse-attention-object-detection)**
   - **Description:** Custom Transformer-based object detector replacing global attention with random sparse sampling.
   - **Impact:** Reduced complexity from $O(N^2)$ to $O(N \cdot K)$. Achieved **68.7% mAP** at 18 FPS (2.6× faster than DETR).
   - **Tech:** Python, PyTorch, Transformers, PASCAL VOC, ResNet-18

5. **[D-NeRF Evaluation for Dynamic Novel View Synthesis](https://github.com/saddam1838/dnerf-view-synthesis-eval)**
   - **Description:** Comprehensive quantitative evaluation of D-NeRF on 8 non-rigid motion scenes using official PyTorch pretrained models.
   - **Impact:** Mapped performance drops in high-velocity, articulated motion scenarios via extensive ablation studies.
   - **Tech:** Python, PyTorch, CUDA, 3D Vision

6. **[Hybrid ML Framework for Smart Grid Cybersecurity](https://github.com/saddam1838/smart-grid-fdia-detection)**
   - **Description:** Hybrid ML pipeline for False Data Injection Attack (FDIA) detection utilizing PCA and Stacked Autoencoders.
   - **Impact:** Achieved **99.32% accuracy** with Random Forest after addressing severe class imbalance.
   - **Tech:** Python, Scikit-learn, PCA, Stacked Autoencoders
